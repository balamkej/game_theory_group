## Key for `sentences.csv`

### cons (constructions)

+ `main` -- main clauses
+ `rf` -- rigid or flexible. 0 for rigid, 1 for flexible.
+ `q1`/`q2` -- The first or second (linear) quantifier. Code as the actual word. We can do coding for universal or existentials later.
